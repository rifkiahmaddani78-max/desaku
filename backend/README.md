# Desa Ku - Backend API

Sistem Informasi Desa Modern berbasis Node.js + Express + MySQL

- Authentication dengan JWT (Access + Refresh Token)
- Role Based Access Control (Admin & Kepala Keluarga)
- Manajemen Data Wilayah (Dusun → RW → RT)
- Manajemen Data Penduduk
- Sistem Bantuan Sosial (Bansos)
- Berita & Jadwal Desa
- Security Best Practices
